Инструкция по настройке автоматического копирования после сборки
=============================================================

Для автоматического копирования файлов в папку Revit 2021 после сборки проекта, выполните следующие шаги:

1. В Visual Studio щелкните правой кнопкой мыши по проекту UNI_Tools_AR в Solution Explorer
2. Выберите "Properties" (Свойства)
3. Перейдите на вкладку "Build Events" (События сборки)
4. В поле "Post-build event command line" (Командная строка после сборки) вставьте следующую команду:

   call "$(ProjectDir)CopyToRevit2021.bat"

5. Нажмите "Apply" (Применить) и "OK"

После этого при каждой успешной сборке проекта файлы плагина будут автоматически копироваться в папку:
- "%USERPROFILE%\AppData\Roaming\Autodesk\Revit\Addins\2021\"

Что копируется:
- Все DLL файлы в подпапку UNI_Tools_AR
- Файл .addin в корневую папку Addins
- PDB файлы для отладки в подпапку UNI_Tools_AR
- Конфигурационные файлы в подпапку UNI_Tools_AR

Примечание: Если вам нужно добавить другие версии Revit (например, 2022 или 2023), 
отредактируйте файл CopyToRevit2021.bat, добавив аналогичные строки для других папок.
